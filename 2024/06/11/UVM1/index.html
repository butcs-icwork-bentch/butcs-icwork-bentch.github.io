代码说明

  1. 类定义：

    • my_driver 继承自 uvm_driver，这是UVM中用于驱动DUT的基类
    • 使用 extern 关键字声明 main_phase 任务，表示在类外实现

  2. 构造函数：

    • 设置默认名称为 “my_driver”
    • 父组件默认为 null

  3. main_phase 实现：

    • 初始化信号：将数据线和有效信号置为无效
    • 等待复位：直到复位信号释放
    • 发送数据：循环发送256个随机数据
    • 每个数据发送时：
      • 等待时钟上升沿
      • 生成随机数据
      • 置有效信号
      • 打印调试信息
    • 发送完成后置有效信号为无效

  4. 调试信息：

    • 使用 uvm_info 宏打印调试信息
    • 参数说明：
      • “my_driver”：组件名称
      • “data is drived”：信息内容
      • UVM_LOW：信息级别
